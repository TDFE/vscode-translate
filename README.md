### 安装插件
- 在vscode应用商店搜索 octopus-translate 进行安装

### 设置里面增加配置
``` javascript
"octopus": {
    "language": "en-US", // 非必填 默认zh-CN
    "color": "red" // 非必填,默认“#73935A”
}
```

### 特别注意
- 默认在package.json同级要有.octopus目录，里面为各语种的key/value
