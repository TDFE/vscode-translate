### 安装插件
- 在vscode应用商店 vscode-translate 安装

### 设置里面增加配置
``` javascript
"tongdun": {
    "translation": {
        "language": "zh-CN" // 配置按照那种语言进行显示，如果选择中文就会显示中文，如果改为英文就显示英文
    }
}
```

### 特别注意
- 默认在package.json同级要有.kiwi目录，里面为各语种的key/value
